# hel zapret ui

Portable GUI-приложение для Windows, которое упрощает работу с **zapret-discord-youtube**: установка и обновление `zapret`, запуск и остановка службы, выбор стратегии, игровой режим, пользовательские списки, тесты соединения и PDF-отчёты.

Приложение рассчитано на обычного пользователя: без ручного ковыряния bat-файлов, PowerShell-меню и внутренностей `zapret`.

---

## Что умеет

- включать и выключать `zapret` из главной вкладки;
- устанавливать и обновлять `zapret` прямо из интерфейса;
- запускать как Windows-службу и управлять ей без ручных команд;
- переключать стратегию обхода;
- редактировать пользовательские списки:
  - сайты для обхода;
  - сайты без изменений;
  - IP-адреса без изменений;
- запускать тесты соединения:
  - Standard;
  - DPI checkers;
- сохранять историю прогонов и собирать **PDF-отчёты**;
- работать из системного трея;

---

## Для кого это

Если тебе нужен `zapret`, но не хочется каждый раз:

- искать нужный `.bat`;
- помнить порядок запуска;
- руками править списки;
- разбираться, какая стратегия реально сработала;
- собирать логи по разным папкам,

то `hel zapret ui` берёт это на себя.

---


## Быстрый старт

1. Скачай релиз.
2. Распакуй архив в любую папку.
3. Запусти `hel_zapret_ui.exe`.
4. Подтверди UAC-запрос администратора.
5. Нажми **Установить zapret**, если `zapret` ещё не скачан.
6. Выбери стратегию и включи `zapret`.

---

## Основные возможности

### Главная

- отображает статус `zapret`;
- показывает текущую стратегию;
- даёт включить, выключить, установить или перезапустить `zapret`;
- умеет применять смену стратегии без ручного запуска bat-файлов.

### Настройки → zapret

- переключение игрового режима;
- выбор режима применения игрового фильтра;
- сохранение и применение изменений с перезапуском, если это нужно.

### Настройки → списки

- отдельные компактные таблицы для пользовательских списков;
- кнопки добавления и удаления;
- сохранение списков в конфиге приложения.

### Тесты

- запуск тестов по выбранным стратегиям;
- отслеживание прогресса;
- история запусков;
- PDF-отчёты по результатам.

## Требования

- Windows 10 / 11
- права администратора
- Python 3.11+ для локального запуска без .exe файла и сборки

> При обычном использовании отдельная установка приложения не нужна: это portable-утилита.

---

## Локальный запуск проекта

### 1. Создать виртуальное окружение

```bat
python -m venv .venv
.venv\Scripts\activate
python -m pip install -U pip
pip install -r requirements.txt
```

### 2. Запустить приложение

```bat
python -m src.main
```

Если приложение стартует без прав администратора, оно само запросит повышение прав.

---

## Локальная сборка

### Установка зависимостей для сборки

```bat
python -m venv .venv
.venv\Scripts\activate
python -m pip install -U pip
pip install -r requirements.txt
pip install -r requirements-dev.txt
```

### Сборка

```bat
:: onedir
pyinstaller -y hel_zapret_ui.spec

:: onefile
pyinstaller -y hel_zapret_ui_onefile.spec
```

Результат:

- `dist\hel_zapret_ui\` — папочная сборка `onedir`;
- `dist\hel_zapret_ui.exe` — одиночный exe `onefile`.
